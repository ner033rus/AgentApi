# AgentAPI — клиент для macOS

Нативное приложение **только для menu bar** (без иконки в Dock): компактные **четыре столбца** по данным с **AgentAPI Server**, опрос раз в секунду.

---

## Зачем это нужно

| Цель | Реализация |
|------|------------|
| Видеть нагрузку удалённой/локальной машины с сервером | HTTP `GET /api/v1/metrics` |
| Не занимать место в Dock | `LSUIElement` / политика accessory |
| Подключаться к любому хосту | Сохранённый базовый URL в настройках |

Клиент **ничего не меряет сам** — все цифры с той системы, где запущен [Server](../Server/README.md).

---

## Требования

- **macOS 13** (Ventura) или новее  
- **Swift 5.9+** (Command Line Tools или Xcode)  
- Сборка под архитектуру вашего Mac (**Apple Silicon** или Intel — `swift build` собирает под текущую машину)

---

## Сборка

### Только бинарник

```bash
cd ClientMacOS
swift build -c release
```

Исполняемый файл: `.build/release/AgentAPIMenuBar`

### Пакет `.app`

```bash
cd ClientMacOS
chmod +x build-app.sh
./build-app.sh
```

Появится каталог **`AgentAPIMenuBar.app`** — его можно перенести в `/Applications` и запускать оттуда.

```bash
open AgentAPIMenuBar.app
```

---

## Использование

1. Запустите **сервер** AgentAPI (см. [Server/README.md](../Server/README.md)), доступный по сети с вашего Mac.
2. Запустите клиент — в строке меню появятся четыре столбика.
3. Чтобы сменить адрес сервера: **клик по иконке → «Адрес сервера…»** и введите **базовый URL** без пути к API, например:
   - локально: `http://127.0.0.1:8765`
   - в LAN: `http://192.168.1.10:8765`

Клиент сам запрашивает `{baseURL}/api/v1/metrics`.

Адрес хранится в **UserDefaults** приложения (ключ `AgentAPI.serverBaseURL`). Значение по умолчанию в коде: **`http://127.0.0.1:8765`**.

## Что означают столбцы (слева направо)

| № | Цвет | Смысл |
|---|------|--------|
| 1 | Голубой | **CPU**, % загрузки |
| 2 | Зелёный | **RAM**, % использования |
| 3 | Оранжевый | **GPU util**, % (если на **сервере** есть NVIDIA и `nvidia-smi`). Иначе столбец низкий/бледный — «GPU с API не доступна», а не обязательно 0% |
| 4 | Фиолетовый | **Доля RAM под Ollama** — сумма памяти процессов Ollama на сервере относительно общего объёма RAM (0–100%) |

Если справа сверху видна **красная точка** — запрос к серверу не удался (сеть, неверный URL, сервер выключен).

## Структура репозитория (этот каталог)

| Путь | Назначение |
|------|------------|
| `Package.swift` | SwiftPM, исполняемая цель `AgentAPIMenuBar` |
| `Sources/AgentAPIMenuBar/` | Исходники (AppKit, опрос API, отрисовка столбиков) |
| `Resources/Info.plist` | Bundle для `.app` (в т.ч. `LSUIElement`) |
| `build-app.sh` | Сборка release и упаковка в `AgentAPIMenuBar.app` |

## Ограничения

- Используется **HTTP** к серверу; для **HTTPS** укажите URL с `https://` (на сервере должен быть валидный TLS или настройте доверие на Mac при самоподписанных сертификатах).
- Частота обновления фиксирована (около **1 с**); при необходимости интервал можно изменить в коде (`StatusBarController`).
